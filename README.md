# Together We Grow Montessori School Website

Official static website for Together We Grow Montessori School.

## Deployment
This site is designed for Cloudflare Pages.

Build settings:
- Framework preset: None
- Build command: leave blank
- Build output directory: leave blank or `/`

## Editing notes
- Homepage content: `index.html`
- Main design styles: `styles.css`
- Small animations/menu behavior: `script.js`
- Images and logo files: `assets/`

## Common updates
- Replace the principal photo placeholder: add the portrait to `assets/`, then update the placeholder in `index.html`.
- Add teacher profiles: copy one `.team-card` block in the Team section and replace the name, title, and text.
- Add gallery images: replace `.gallery-tile` placeholders with image cards.
